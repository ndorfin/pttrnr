# PTTRNR

A pattern library generator

## Dependencies

- Node.js (v16.10+)
- NPM (v7.24+)

## Commands

- `npm i`: Install
- `npm run build`: Run a production build
- `npm run build:debug`: Run a production build in DEBUG mode
- `npm run serve`: Run a dev server
- `npm run serve:debug`: Run a dev server in DEBUG mode
